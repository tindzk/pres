  • Reaktive Datenströme

Reaktive Datenströme

  • Statt mutable Variablen/Arrays:
    • Channel: Strom von Werten
    • Aggregate: Strom von Listenänderungen
  • Kombinatoren wie map(), filter(), flatMap(), zip(), etc.

Beispiel


val todo = Channel[String]()
todo.map(_.trim)
    .filter(_.nonEmpty)
    .foreach(println)

Input.Text()
  .placeholder("What needs to be done?")
  .bind(todo)

Two-way-Binding


val todo = Channel[String]()

Input.Text().bind(todo, live = true)
Input.Text().bind(todo, live = true)
